隐私政策

Med Tracker 移动应用

开发者：格列布·安德烈耶夫（以下简称“我们”或“我方”）。

1. 基本原则与运行模式

Med Tracker 应用（以下简称“本服务”）以用户隐私保护为最高优先级进行开发。本隐私政策旨在说明我们处理用户数据的相关做法。

本服务的核心原则如下：

  • 无服务器架构：本服务完全在本地运行，不使用远程服务器、云存储或任何第三方 API 存储用户数据。
  • 无需注册登录：我们不要求且不提供用户账号注册或任何形式的身份验证功能。
  • 无数据传输：用户数据绝不会被收集、处理或传输至设备以外的任何地方（包括我方或任何第三方）。

2. 信息收集与存储

2.1 我们不会收集的数据

我们不会收集或存储以下类型的数据：

  • 任何个人身份信息（PII），如姓名、电子邮箱地址或电话号码；
  • 财务相关信息；
  • 任何个人健康信息（PHI）或与您健康状况相关的医疗数据。
2.2 本地存储数据（用户数据）

您在 Med Tracker 应用中输入的所有药品信息，仅存储于您的移动设备本地数据库中。我们无法访问这些数据。

本地存储的用户数据包括：

  • 药品名称及简要说明（由用户输入）；
  • 药品类型（由用户从预设列表中选择）；
  • 有效期；
  • 用户添加的其他备注信息。
error重要提示

由于所有数据均仅存储于设备本地，若您卸载应用、重置设备或更换新设备，我们将无法为您恢复数据。

2.3 第三方服务

我们未集成任何第三方分析、广告或其他数据收集服务。本应用仅使用操作系统提供的、实现本地功能所必需的最小服务。

3. 权限与通知

为实现核心功能（药品有效期提醒），Med Tracker 会向操作系统申请本地通知权限。

4. 儿童隐私保护

由于 Med Tracker 应用不收集任何个人身份信息，且未设置年龄限制，因此符合中国及国际关于“不收集儿童个人信息类应用”的隐私保护要求。

5. 隐私政策的变更

我们可能不定期更新本隐私政策。更新后的版本将发布于本页面。建议您定期查阅本政策。
